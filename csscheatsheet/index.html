CSS Cheatsheet

CSS Properties (Свойства)

Color and Background

  • color - Сменя цвета на текста
  • background-color - Сменя цвета на фона
  • background-image - Променя фона да е снимка
  • background-size - Променя размера на фона
  • opacity - Променя наситеността на елемента

Text and Font

  • font-family - Променя шрифта на текста
  • font-size - Променя размера на шрифта на текста
  • font-weight - Променя дебелината на шрифта на текста
  • line-height - Променя разстоянието между редовете
  • text-align - Променя центрирането на текста
  • text-decoration - Променя украската на текста (подчертаването на линковете например)
  • text-transform - Променя капитализацията на текста
  • letter-spacing - Променя разстоянието между буквите
